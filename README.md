# 贴吧无线UI Action生成器

## 开始之前

在开始使用前，请确保你已经全局安装以下组件：
- [bower](http://bower.io/) `npm install -g bower`
- [grunt-cli](http://gruntjs.com/) `npm install -g grunt-cli`

## 安装

首先安装yeoman

```bash
npm install -g yo
```

然后安装本生成器

```bash
npm install -g generator-uiaction
```

## 使用

### 生成action

在UI模块（如mowsenior）根目录执行：

```bash
yo uiaction
```
然后根据向导来一步一步操作，再也不用复制文件了
